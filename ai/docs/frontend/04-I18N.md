# 04-I18N.md — Anthekira.dev

## 1. Configuración
**next-intl** con routing localizado. Plugin en `next.config.ts` → `./frontend/src/lib/i18n.ts`.

```typescript
// frontend/src/lib/routing.ts
export const routing = defineRouting({
  locales: ['es', 'en', 'pt'],
  defaultLocale: 'es',
  localePrefix: 'as-needed',  // / → español, /en → inglés
  localeDetection: true,
});
```

## 2. Archivos de Traducción
```
public/locales/{es,en,pt}.json
```
Estructura por namespaces: common, nav, hero, about, skills, technologies, projects, services, methodology, ai_native, contact, footer, admin.

**Nota:** Contenido dinámico del CMS (proyectos, servicios) NO está en estos JSON. Se almacena en BD y se consulta desde Supabase.

## 3. Routing Localizado
| URL | Comportamiento |
|---|---|
| `anthekira.dev` | Detecta Accept-Language → / (es), /en, /pt |
| `anthekira.dev/en` | Landing en inglés |
| `anthekira.dev/admin` | Admin en español (sin prefijo idioma) |

Middleware: next-intl + Supabase auth para rutas admin.

## 4. Uso en Server Components
```tsx
import { getTranslations } from 'next-intl/server';
const t = await getTranslations('hero');
// t('title'), t('subtitle')

// Metadata SEO por idioma
export async function generateMetadata({ params: { locale } }) {
  const t = await getTranslations({ locale, namespace: 'projects.meta' });
  return { title: t('title'), description: t('description'), alternates: { languages: { es: '/es/projects', en: '/en/projects', pt: '/pt/projects' } } };
}
```

## 5. Contenido Dinámico del CMS
Auto-traducción DeepL al guardar (no en cada visita):
```
Admin escribe en ES → Guarda en BD → DeepL API → EN + PT → Tablas _translations
```
Las tablas de traducción (`project_translations`, `service_translations`, etc.) usan `content JSONB` + `translation_status`.

## 6. Estrategias
| Tipo | Método | Almacenamiento |
|---|---|---|
| UI (menús, botones) | next-intl JSON | public/locales/*.json |
| SEO (metadatos) | next-intl JSON | public/locales/*.json |
| Contenido CMS (proyectos, servicios) | DeepL auto-traducción | Tablas _translations en BD |
| Panel admin | Solo español | — |
